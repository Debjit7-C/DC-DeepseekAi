🚀 DeepSeekAI Clone (MERN Stack)

A full-stack AI-powered web application inspired by DeepSeekAI, built using the MERN stack (MongoDB, Express.js, React.js, Node.js). This project delivers a smooth and interactive user experience for generating real-time AI responses with a scalable backend architecture.

📌 Features
⚡ Real-time AI response generation
💬 Interactive and responsive user interface
🔐 Secure API handling and backend integration
📊 Efficient data storage using MongoDB
🌐 RESTful APIs for seamless communication
📱 Fully responsive design
🛠️ Tech Stack
Frontend: React.js
Backend: Node.js, Express.js
Database: MongoDB
Other Tools: Axios, REST APIs, Git
📂 Project Structure
DeepSeekAI_Clone/

├── server/        # Node.js + Express backend

└── README.md
⚙️ Installation & Setup
Clone the repository
git clone <your-github-link>
cd DeepSeekAI_Clone
Install dependencies

Frontend:

cd client
npm install

Backend:

cd server
npm install
Run the project

Backend:

npm start

Frontend:

npm start
🔑 Environment Variables

Create a .env file in the server folder and add:

PORT=5000
MONGO_URI=your_mongodb_connection
API_KEY=your_ai_api_key
